`npm run dev` boots three processes in parallel: the Python TTS server (`.venv/bin/python scripts/tts_server.py`), the Node dev chat proxy (`node scripts/dev-api-server.mjs` on port 8791), and the Vite dev server (port 8787). The Vite dev proxy forwards `/api/chat` to the Node mirror and `/api/tts*`/`/api/fetch-url` to the Python TTS server. For production, `api/*.js` are deployed as Vercel Serverless Functions and `vercel.json` configures their runtime.
