`scripts/dev-api-server.mjs` listens on `127.0.0.1:8791` and requires `GEMINI_API_KEY*` or `DEEPSEEK_API_KEY` env vars. `scripts/tts_server.py` binds `127.0.0.1:8790` and needs `edge_tts` installed. `scripts/tts_regression.mjs` expects both the Vite dev server (proxying `/api/tts` on `:8787`) and the local TTS server running, then executes itself via `node scripts/tts_regression.mjs`. The legacy PWA is served with `cd _legacy_flat && python3 -m http.server 8765`.
