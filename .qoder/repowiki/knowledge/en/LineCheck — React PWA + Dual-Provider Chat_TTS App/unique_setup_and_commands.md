`npm run dev` starts three processes in parallel: the Python TTS server (`scripts/tts_server.py` on 8790), the Node dev API mirror (`scripts/dev-api-server.mjs` on 8791), and the Vite dev server (8787) which proxies `/api/*` to them. Requires a Python `.venv` with `scripts/tts_server.py`'s dependencies installed. For production on Vercel, set `GEMINI_API_KEY[_N]` and/or `DEEPSEEK_API_KEY` as environment variables; locally you may also paste a key into Settings.
